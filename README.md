# CeloPoly 🎲💎

**The money game you play on-chain. Built on Celo.**

CeloPoly is a fully decentralized, EVM-compatible multiplayer board game engineered to teach real-world financial literacy through engaging, high-stakes gameplay. By leveraging the Celo blockchain, CeloPoly transforms classic property trading mechanics into a dynamic Web3 experience with real financial consequences and rewards.

## 🌟 Key Features

- **On-Chain Escrow & Prize Pool**: Players stake `1 cUSD` to join a table. The smart contract acts as an immutable escrow, locking funds securely.
- **Provably Fair Mechanics**: Dice rolls, property acquisitions, and card draws are managed entirely via verifiable smart contract logic.
- **Real Yield & Rent**: Buy properties, collect rent in cUSD, and stack houses on your color sets to bankrupt opponents.
- **Winner-Takes-All Economy**: The last player standing claims the entire accumulated prize pool directly to their wallet.
- **Native MiniPay Integration**: Seamlessly integrated with Opera MiniPay for ultra-low friction onboarding, allowing users to connect, stake, and play instantly within the MiniPay ecosystem using stablecoins.
- **Dynamic Event Streaming**: Powered by `wagmi` and `viem`, the React/Vite frontend listens to live blockchain events for real-time multiplayer synchronization.
- **Mobile-Optimized**: Designed with a sleek, glassmorphic UI that runs flawlessly on Minipay and mobile Web3 browsers.

## 🏗️ Technical Architecture

- **Smart Contracts**: Written in Solidity. Features modular architecture separating core Game Logic, Leaderboard Tracking, and Trophy NFT Minting.
- **Frontend**: React + Vite + TypeScript.
- **Web3 Integration**: Wagmi v2 hooks for seamless wallet connection, transaction signing, and live event subscription.
- **Styling**: Tailwind CSS with custom glassmorphism, micro-animations, and dynamic visual state management.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- A Celo-compatible wallet (e.g., MetaMask, Minipay)
- Testnet CELO and cUSD for gas and entry fees.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Yilkash/Celopoly.git
   cd Celopoly
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
